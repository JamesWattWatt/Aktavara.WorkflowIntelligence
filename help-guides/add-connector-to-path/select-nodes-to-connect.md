# Select Nodes to Connect

Before creating a connector, you must select two nodes that you want to connect.

## Selection Process

1. **Click on the first node** (start node)
   - The node will be highlighted to show it's selected
   - This is where the connector will begin

2. **Hold Shift and click on the second node** (end node)
   - Or use the "Add Connector" tool first, then select nodes
   - Both nodes should now be selected

3. **Verify the selection**
   - Both nodes should be visibly highlighted
   - Status bar shows "2 nodes selected"

## Valid Connector Paths

You can connect:
- Any node to any other node in the path
- A node to itself (self-loop)
- Multiple connectors to the same node (convergent paths)
- Nodes that are already connected (creates duplicate connector)

## Invalid Connections

The system will prevent you from creating:
- Connectors to nodes in a different path
- Connectors in collapsed node groups (expand first)

## Tips for Successful Connections

- **Organize your path** — Position nodes clearly before connecting
- **Test the flow** — Make sure the connector direction makes logical sense
- **Use labels** — Give connectors meaningful names to describe the transition

## Next Steps

Once you have selected the two nodes:
1. Right-click and choose "Create Connector"
2. Or drag from one node to another
3. Configure the connector properties if needed
4. Save the connector
