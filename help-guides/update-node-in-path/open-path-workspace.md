# Open a Path Workspace

To start editing nodes in a path, you must first open the path workspace.

## How to Open a Path

1. **Search for the path** using the search records feature
   - Use keywords to find the path you want to edit
   - Look for the path's name or ID

2. **Click "Open Workspace"** on the path record
   - This loads the complete path with all nodes and connectors
   - The workspace displays the visual structure of your path

3. **Wait for the workspace to load**
   - Large paths may take a moment to fully load
   - All nodes and connectors will be displayed once loading is complete

## Workspace Components

Once the path workspace is open, you can see:
- **Nodes** — Individual steps or decision points in the path
- **Connectors** — Links between nodes that define the flow
- **Properties panel** — Details and settings for selected elements
- **Toolbar** — Actions like Save, Add Node, Edit, etc.

## What's Next

Once the workspace is open:
1. Select a node to view its properties
2. Modify the node as needed
3. Save your changes to the node
4. Save the entire path workspace when done

## Tips

- Use the search feature in the workspace to quickly find nodes
- Right-click on nodes for context menu options
- Use keyboard shortcuts for faster editing (Ctrl+S to save)
