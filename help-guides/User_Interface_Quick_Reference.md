# User Interface Toolbars and   Shortcuts 

Below is a brief overview of essential menu commands, toolbar actions, and keyboard shortcuts in **Aktavara Console**.

---

## Menu Items

### Ribbon toolbar - Main 

| Name | Shortcut | Action |
|------|----------|--------|
| Open | Ctrl+O | Opens an XML file |
| Save | Ctrl+S | Saves the active workspace |
| Save All | Ctrl+Shift+S | Saves all open workspaces |
| Undo         | Ctrl+Z       | Undo last action                              |
| Redo         | Ctrl+Y       | Redo last undone action                       |
| Refresh      | F5           | Refreshes the workspace                       |
| Print | Ctrl+P | Prints the active workspace |
| Properties | F4 | Opens the Properties window |
| Show in Tree | Ctrl+T | Displays the selected object in Explorer tree |

### Ribbon toolbar - Edit

| Name | Shortcut | Action |
|------|----------|--------|
| Undo | Ctrl+Z | Undo last action |
| Redo | Ctrl+Y | Redo last undone action |
| Add Worksheet | — | Opens a new Worksheet |

### Ribbon toolbar - View

| Name | Shortcut | Action |
|------|----------|--------|
| Explorer | — | Opens the Explorer window |
| Messages | — | Opens the Messages window |
| Progress | — | Opens the Progress window |
| Properties | F4 | Opens the Properties window |
| Shapes | — | Opens the Shapes window |
| Information | — | Opens the Information window |
| Record Searches | — | Opens the Record Searches window |

### Ribbon toolbar - Tools

| Name | Shortcut | Action |
|------|----------|--------|
| Connections | — | Opens the Connections workspace |
| Routing | — | Opens the Routing workspace |
| Path Creation | — | Opens the Path Creation workspace |
| Add-In | — | Opens the Add-In module |

### Ribbon toolbar - Window

| Name | Shortcut | Action |
|------|----------|--------|
| Use Tabbed MDI | — | Toggles tabbed workspace behavior |
| Cascade | — | Cascade open workspaces |
| Tile Horizontal | — | Tile open workspaces horizontally |
| Tile Vertical | — | Tile open workspaces vertically |
| Close All Workspaces | — | Closes all opened workspaces |

### Info Sheet

| Name          | Shortcut | Action                                |
| ------------- | -------- | ------------------------------------- |
